# 🧘 ZenPose – Yoga Pose Detection System

ZenPose is an AI-powered web application that detects and analyzes yoga poses in real-time using computer vision and machine learning. It helps users improve posture accuracy and track their yoga performance.

---

## 🚀 Features

- 🔐 OTP-based authentication system
- 🎥 Real-time pose detection using MediaPipe
- 🤖 Machine Learning model for pose classification
- 📊 Pose accuracy tracking and history
- 💾 SQLite database integration
- 🌐 Interactive frontend with HTML, CSS, and JavaScript

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Flask (Python)
- **Database:** SQLite
- **Machine Learning:** Scikit-learn
- **Computer Vision:** MediaPipe, OpenCV
